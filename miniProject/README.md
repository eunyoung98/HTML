## About The Project

<strong>송파구 맛집 사이트:bento:</strong>는 송파구 근처 맛집, 특히 경찰병원역에서 도보로 방문 가능한 맛집들을 소개 했습니다! <br>
한식 / 양식 / 중식 / 일식 별로 대표 맛집을 추천해드리고 있습니다:thumbsup: <br>
카테고리별로 페이지의 기능들이 다르니 하나씩 재미있게 구경해주세요!!:blush:
